📝 توضیحات تکنیکال:

  • Tactical: الگوی شبکه نظامی دوگانه - یک grid درشت 10x10 با رنگ accent و یک grid ریز 3x3 برای بافت. استفاده از mix-blend-mode برای ترکیب بهتر.
  • Glass: افکت شیشه مات واقعی با backdrop-filter و چند gradient شعاعی برای شبیه‌سازی شکست نور. Inset shadows برای عمق سه‌بعدی.
  • Notification: سه gradient شعاعی با انیمیشن pulse برای جلب توجه. Multiple inset shadows برای ایجاد هاله درخشان. Border چپ ضخیم‌تر برای تاکید.
  • Data: خطوط اسکن افقی با الگوی پیچیده 5 پیکسلی که شبیه مانیتورهای CRT است. انیمیشن sweep برای حرکت scanline و inset shadow برای glow فسفری.
  • Minimal: دو الگوی diagonal متقاطع (45 و -45 درجه) با opacity بسیار کم برای ایجاد بافت ظریف فیلم. Filter contrast برای وضوح بیشتر.
