# گزارش تغییرات پالت رنگی RHUDS Pro

## خلاصه تغییرات

تمام رنگ‌های پیش‌فرض پروژه RHUDS Pro با پالت رنگی HUD جدید جایگزین شدند.

## پالت رنگی جدید

### 5 رنگ اصلی HUD
1. **#DE41F2** - صورتی/ارغوانی روشن (Primary)
2. **#47038C** - بنفش (Secondary)  
3. **#130226** - بنفش خیلی تیره (Background)
4. **#342373** - بنفش تیره (Surface/Info)
5. **#400E29** - قرمز تیره (Error)
