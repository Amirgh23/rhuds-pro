## 🔄 Migration Path

### From Chart.js

```typescript
// Before
const ctx = document.getElementById('myChart').getContext('2d');
const myChart = new Chart(ctx, { type: 'line', data: {...} });

// After
const chart = new Chart({ type: 'line', data: {...} });
chart.render('#myChart');
```

### From Arwes

```jsx
// Before
<Frame>
  <FrameBox>Content</FrameBox>
</Frame>;

// After
const Frame = generator.createFrame('advanced');
<Frame>Content</Frame>;
```
