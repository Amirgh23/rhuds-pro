## نحوه استفاده

### در تم‌ها
```typescript
const theme = {
  colors: {
    primary: '#29F2DF',      // Cyan - برای دکمه‌ها، لینک‌ها
    secondary: '#1C7FA6',    // Blue - برای عناصر ثانویه
    background: '#0A1225',   // Dark - پس‌زمینه اصلی
    surface: '#28125A',      // Purple - کارت‌ها و سطوح
    accent: '#EF3EF1',       // Pink - تاکید و هشدار
  }
}
```

## نکات مهم

1. **Toast ها**: رنگ‌های Toast تغییر نکرده‌اند (success=سبز، error=قرمز، warning=نارنجی)
2. **رنگ‌های خنثی**: سفید (#FFFFFF) و خاکستری (#888888) برای متن استفاده می‌شوند
3. **شفافیت**: از rgba برای شفافیت استفاده کنید (مثلاً 0.3 برای بوردر، 0.8 برای پس‌زمینه)
