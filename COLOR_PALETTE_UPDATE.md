# پالت رنگی جدید HUD

## رنگ‌های اصلی پروژه

پالت رنگی کل پروژه RHUDS Pro به رنگ‌های زیر تغییر یافته است:

### رنگ‌های HUD اصلی
1. **Primary (اصلی)**: `#DE41F2` - صورتی/ارغوانی روشن
2. **Secondary (ثانویه)**: `#47038C` - بنفش
3. **Background (پس‌زمینه)**: `#130226` - بنفش خیلی تیره
4. **Surface (سطح)**: `#342373` - بنفش تیره
5. **Error (خطا)**: `#400E29` - قرمز تیره

### تغییرات انجام شده

#### 1. فایل‌های تم اصلی
- ✅ `packages/core/src/theme/themes.ts` - تمام تم‌ها به‌روز شدند
- ✅ `packages/core/src/theme/creators.ts` - رنگ‌های پیش‌فرض تغییر کردند
- ✅ `packages/demo-app/src/App.tsx` - رنگ‌های اپلیکیشن دمو به‌روز شدند
- ✅ `packages/demo-app/src/index.css` - رنگ اسکرول‌بار تغییر کرد

#### 2. تم‌های موجود
- **dark**: Primary=#DE41F2, Secondary=#47038C, Background=#130226
- **neon-purple**: Primary=#DE41F2, Secondary=#47038C
- **neon-violet**: Primary=#47038C, Secondary=#342373
- **neon-crimson**: Primary=#400E29, Secondary=#DE41F2

| رنگ قدیمی | رنگ جدید | کاربرد |

### نکات مهم

1. **رنگ‌های خنثی**: رنگ‌های خاکستری و سفید برای متن همچنان استفاده می‌شوند
2. **سازگاری**: تمام کامپوننت‌ها به صورت خودکار از تم جدید استفاده می‌کنند
3. **دمو**: صفحات Showcase، Playground و Docs همگی با پالت جدید نمایش داده می‌شوند

### دستور اجرا

برای مشاهده تغییرات:

```bash
npm install
npm run dev
```

سپس به آدرس `http://localhost:5173` بروید.
