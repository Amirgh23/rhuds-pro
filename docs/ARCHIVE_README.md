# Documentation Archive

این پوشه شامل اسناد قدیمی و تاریخی پروژه است.

## نکات مهم:

- اسناد فعلی در root directory قرار دارند
- برای اطلاعات فعلی، `README.md` و `START_HERE.md` را بخوانید
- اسناد قدیمی فقط برای مرجع تاریخی نگهداری شده‌اند

## ساختار:

- `charts/` - اسناد مربوط به سیستم نمودارها
- `coldwar/` - اسناد مربوط به Cold War redesign
- `phases/` - اسناد مربوط به فازهای پروژه
- `other/` - سایر اسناد
